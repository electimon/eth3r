**This tool is currently a work-in-progress**



Instructions

1. Prepare the offerings and take a shower

2. Download the latest version from my releases page

3. cd into the project location

4. Open the program with ./Eth3r-Linux

5. Follow all instructions, and wait for the IPSW to generate.

6. Restore to the IPSW in iTunes, and use xpwndfu to boot up after the restore
